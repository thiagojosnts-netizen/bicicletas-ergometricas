# One Light Portrait

> Task ID: mcnally-one-light-portrait
> Agent: Joe McNally (Photography Lighting Expert)
> Version: 1.0.0

## Description

Create professional portrait lighting using a single light source. Based on Joe McNally's "One Light Methodology" - mastering one perfect light before adding complexity. This task helps photographers achieve professional results with minimal equipment.

## Prerequisites

- One light source (speedlight, strobe, or continuous)
- At least one modifier (umbrella, softbox, or bounce surface)
- Light stand or means to position light off-camera
- Trigger or sync method

## Workflow

### Interactive Elicitation

1. **Identify Available Light**
   - What type of light do you have? (speedlight, strobe, continuous)
   - What modifiers do you have? (umbrella, softbox, reflector)
   - Can you position the light off-camera?

2. **Define Portrait Style**
   - What mood are you going for? (dramatic, natural, beauty, corporate)
   - Subject characteristics (age, gender, skin tone)
   - Final use (social media, print, corporate, personal)

3. **Assess Environment**
   - Shooting indoors or outdoors?
   - Available ambient light level
   - Bounce surfaces available (walls, ceilings)?

### Steps

1. **Select One Light Pattern**
   - Choose based on desired mood and subject:
     * **Window Light Simulation**: Large modifier, 45 degrees, natural feel
     * **Rembrandt Dramatic**: 45/45 position, triangle on shadow cheek
     * **Beauty Clamshell**: Light above + reflector below
     * **Edge/Rim Dramatic**: Light behind, subject in shadow
     * **Bounce Master**: Indirect light via ceiling/wall
   - Validation: Pattern selected matches mood objective

2. **Position The Light**
   - Set horizontal angle (typically 30-45 degrees from camera axis)
   - Set vertical angle (typically 15-45 degrees above eye level)
   - Position at appropriate distance for desired quality
   - Validation: Light position creates intended shadow pattern

3. **Configure Modifier**
   - Attach appropriate modifier
   - Position modifier at correct angle to subject
   - Adjust distance (closer = softer, farther = harder)
   - Validation: Quality of light matches objective

4. **Set Initial Power**
   - Start at middle power (1/4 or 1/8)
   - Take test shot
   - Adjust based on exposure
   - Validation: Subject properly exposed

5. **Fine-Tune Position**
   - Check catchlights in eyes
   - Verify shadow placement (nose shadow direction)
   - Adjust height for optimal shadow pattern
   - Validation: Catchlights visible, shadows flattering

6. **Balance With Ambient (Optional)**
   - Meter ambient light
   - Decide ratio: flash dominant or ambient fill
   - Adjust shutter speed to control ambient
   - Validation: Flash and ambient work together

7. **Apply One Light Checklist**
   - Run through quality checks
   - Make final adjustments
   - Document the setup
   - Validation: All checklist items pass

- **Setup Diagram**: Single light position documented
- **Camera Settings**: ISO, aperture, shutter, WB
- **Expected Result**: Description of final look
- **Adjustment Guide**: How to modify for variations

## One Light Patterns Reference

### Pattern 1: Window Light Simulation
- **Position**: 45° side, slightly above
- **Modifier**: Large softbox or shoot-through umbrella
- **Character**: Soft, directional, forgiving
- **Best for**: Natural portraits, corporate, beauty

### Pattern 2: Rembrandt Dramatic
- **Position**: 45° side, 45° above
- **Modifier**: Medium source
- **Signature**: Triangle of light on shadow cheek
- **Best for**: Artistic portraits, masculine subjects

### Pattern 3: Beauty Clamshell
- **Position**: Directly above + reflector below
- **Modifier**: Softbox or beauty dish
- **Character**: Even, flattering, open shadows
- **Best for**: Beauty, makeup, glamour

### Pattern 4: Edge/Rim Dramatic
- **Position**: Behind or to side-behind subject
- **Modifier**: Gridded or narrow source
- **Character**: High contrast, mysterious
- **Best for**: Silhouettes, dramatic editorial

### Pattern 5: Bounce Master
- **Position**: Flash aimed at ceiling/wall
- **Modifier**: None (surface is modifier)
- **Character**: Ambient-like, soft, unobtrusive
- **Best for**: Events, candid, natural look

## Success Criteria

- [ ] Single light creates professional-quality image
- [ ] Pattern matches intended mood
- [ ] Catchlights properly positioned
- [ ] Shadow pattern is flattering
- [ ] Setup can be replicated
- [ ] Additional light proven unnecessary

## Error Handling

- **Shadows too harsh**: Move light closer OR use larger modifier
- **Shadows too soft**: Move light farther OR use smaller modifier
- **No catchlights**: Adjust light angle to reflect in eyes
- **Unflattering shadows**: Lower the light or change angle
- **Background too dark**: Move subject closer to background OR add ambient

## McNally Principles Applied

- "One perfect light beats five mediocre ones."
- "If one light isn't working, it's probably in the wrong place."
- "Start simple. One light. One subject. One modifier."
- "Before adding another light, adjust this one."
